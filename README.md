## Getting Started

Clone o repositório:

```bash
git clone git@github.com:LucasDiello/barber-shop.git
```

Entre na pasta do projeto e instale os pacotes:

```bash
npm install
```

Inicialize a aplicação:

```bash
npm run dev
```
